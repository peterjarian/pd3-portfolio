# 8-bit Sound Files

This directory contains 8-bit sound effects for the portfolio.

## Required Sound Files:
- `button-click.wav` - Button click sound
- `button-hover.wav` - Button hover sound
- `navigation.wav` - Page navigation sound
- `start-game.wav` - Start game button sound

## Resources for 8-bit sounds:
- [sfxr.me](https://sfxr.me) - Generate retro game sounds
- [freesound.org](https://freesound.org) - Free sound effects library
- [jsfxr](https://sfxr.me/) - Browser-based sound generator

## Instructions:
1. Visit sfxr.me
2. Generate sounds with these settings:
   - Button Click: Use "Pickup/Coin" preset
   - Button Hover: Use "Blip/Select" preset
   - Navigation: Use "Powerup" preset
   - Start Game: Use "Jump" preset
3. Export as WAV format
4. Keep file sizes under 50KB each
