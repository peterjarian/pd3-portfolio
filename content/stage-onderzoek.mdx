## Wat vind ik belangrijk?

- Ik wil graag stagelopen bij een groot bedrijf.
- Ik wil stagelopen op een bedrijf met een eigen IT-afdeling.
- Ik wil niet werken aan de freelance kant.
- Ik wil werken in een team waar er wordt gefocust op de inhoud.
- Ik wil werken in de buurt van Den Haag.
- Ik wil stagelopen als fullstack product developer aan de web en mobiele kant.

## Waar kan ik een stageplek vinden?

- **Eigen netwerk**: ik heb een heel groot netwerk met veel connecties naar de zakenwereld. Mijn familie is heeft vrij veel invloed binnen sectoren zoals de vliegtuigmaatschappij.
- **LinkedIn**: Ik kan voor de rest zoeken naar stageplekken op mijn LinkedIn mocht mijn eigen netwerk te kort komen.

## Concrete plekken

- **Boskalis**: via een familievriend kan ik hier terecht.
- **Corendon**: mijn oom is lang CEO geweest van dit bedrijf en hier heb ik openingen.
- **NS**: een andere vriend van mijn moeder werkt in het ICT-team van NS.
- **Conlusion**: Een groot bedrijf dat van alles doet met ICT, zowel aan de technische als organisatorische kant. Via een vriend van mijn moeder.
